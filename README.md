#  Automação de Cortina com Sensor LDR

Projeto de automação utilizando Arduino e Python para controle automático de cortinas baseado na luminosidade do ambiente.

##  Objetivo
Desenvolver uma solução que reaja às condições de iluminação, abrindo ou fechando a cortina automaticamente para melhorar conforto e eficiência energética.

##  Tecnologias Utilizadas
- Arduino
- Python
- Sensor LDR (Light Dependent Resistor)
- Comunicação serial

##  Funcionamento do Sistema
1. O sensor LDR mede a intensidade da luz ambiente.
2. O Arduino interpreta os valores captados.
3. O sistema decide automaticamente abrir ou fechar a cortina.
4. A ação é executada sem intervenção manual.

##  Conceitos Aplicados
- Integração hardware + software
- Automação baseada em sensores
- Sistemas embarcados
- Lógica de decisão em tempo real
- Introdução à IoT
